## 🔍 文献内容概览

### 核心研究发现
1. **绿豆粉丝**：GI仅28，是优秀的低GI选择
2. **山药**：GI 52，低GI食物
3. **薏米**：GI 55，低GI食物  
4. **糙米**：GI 82，高GI食物
5. **传统中国食物GI范围广泛**：从低GI(28)到高GI(85)

### 文献分类
- **核心研究** (4篇)：基础GI值测定
- **特定食物研究** (3篇)：山药、馒头等
- **流行病学研究** (3篇)：与疾病关系
- **方法学研究** (2篇)：GI测定方法
- **临床干预研究** (3篇)：实际应用
- **亚洲食物比较研究** (2篇)：区域对比
- **其他研究** (2篇)：相关领域

## 📝 如何使用文献库

### 1. 阅读文献摘要
所有文献摘要都以Markdown格式保存在 `summaries/` 目录，你可以：
- 直接阅读了解研究概况
- 在"关键发现"和"启示"部分添加个人笔记
- 在"笔记"部分记录个人想法

### 2. 查看文献元数据
元数据文件包含完整文献信息，包括：
- 作者、年份、期刊
- 摘要内容
- PubMed链接
- 添加日期

### 3. 下载PDF全文
由于版权限制，PDF需要手动下载：
1. 访问文献的PubMed链接
2. 查找"Full Text"或"PDF"链接
3. 下载PDF到 `pdfs/` 目录
4. 建议命名：`{PMID}.pdf`

### 4. 使用文献索引
索引文件帮助你快速查找：
- 按类别查找文献
- 按年份查找文献
- 了解文献库概况

## 🔄 文献库维护

### 添加新文献
1. 编辑 `init_library.py` 中的 `literature_list`
2. 添加新的文献信息
3. 重新运行初始化脚本

### 更新文献信息
1. 修改 `summaries/` 中的Markdown文件
2. 添加个人笔记和关键发现
3. 更新对糖尿病管理的启示

### 备份文献库
建议定期备份：
```bash

tar -czf literature_backup_$(date +%Y%m%d).tar.gz literature/
```

- 晚餐：少量主食 + 高纤维食物
